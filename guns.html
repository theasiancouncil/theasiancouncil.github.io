If you've ever wondered about what all of the gun varients and stuff are, then you will no longer :D

 

 

 

 

 

 

 

 

 

 

 

 

 

 

 

 

 

 

 

 

 

 

 

 

Assault class uses SMGs and shotguns. The Automatico M1918 Trench/Storm or the Hellriegel 1915 are the best SMGs, for shotguns it is preference.

The spread of the shotguns are:   Sweeper > Factory > Hunter > Slug

 

 

 

 

 

 

 

 

 

 

 

 

 

 

 

 

 

 

 

 

 

Medic classes use SLR (self loading rifles) with a few with full automatic firing modes. The Selbstlader M1916 marksmen is very good, with the Mondragon and Autoloading 8 extended next.

 

 

 

 

 

 

 

 

 

 

 

 

 

 

 

 

 

 

 

 

 

 

The support class uses LMGs, some have tripods that will increase performance when prone. Some of them have Semi-Automatic Firing modes. Here the BAR M1918 does the most damage.

 

 

 

 

 

 

 

 

 

 

 

 

 

 

 

 

 

 

 

 

 

 

 

The Scout class uses all Bolt Action Sniper Rifles (SR) except for the M1903 Experimental. Refer to the scout class guide for the weapon you want to choose, as the sweet spot range is the only one that matters.
